## Part 4: Randomness in Outputs (Temperature)

## Part 6: What Counts as a Prompt

### Exercise 6: Prompt Components Awareness

Everything in a prompt influences the output, including:
- Roles  
- Formatting  
- Constraints  
- Examples  

#### Prompt Comparison

**vs**

3-day project deadline extension.

#### Conclusion
